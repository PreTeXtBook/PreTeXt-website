    Section 1.7 Tables

    Although a WeBWorK problem written in PreTeXt source can’t have a “table” (which would be captioned and cause trouble with free-standing PG output), they can have a “tabular”. Tabulars can be made using the existing syntax and options that PTX offers for table-making, with some exceptions. As of summer 2017, the exceptions are that individual cells cannot have bottom border attributes and individual columns cannot have top border attributes. Also, while PTX table borders are generally minor, medium, or major, when these borders are used in a WeBWorK problem that is rendered as a hard copy problem from within WeBWorK, the only thickness option corresponds to minor. Several more features (like rules in general) do not carry through to static output (print or HTML-preview). However we hope this is only temporary.
