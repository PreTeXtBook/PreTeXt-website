Since PreTeXt was originally called MathBook XML, you will not be surprised to learn that it has robust support for mathematical formulas. Inside the tags that delimit math environments, your code is basically LaTeX , with the caveat that you must be careful with < and & since they are special symbols for XML. When typing math in your PreTeXt code, use \lt for <, use \gt for > (not strictly necessary, but good for symmetry), and use \amp for &.
