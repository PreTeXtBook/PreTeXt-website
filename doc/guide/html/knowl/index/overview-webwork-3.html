It is possible to author WeBWorK automated homework problems directly within your source. A static version will be rendered in your LaTeX/PDF/print output, and a “live” version will be rendered into HTML output (though a student is not able to authenticate against a course). However, you can also extract all of the WeBWorK questions from a textbook into a single archive suitable for uploading into a traditional WeBWorK server. This is a big topic, so see the dedicated Chapter 7 for details.
