Our example here just illustrates using TikZ to make a simple figure (the Hasse diagram of a poset), but lots of other LaTeX graphics packages can be used. One step required is to put the following three lines in the <docinfo> tag of your main file. <latex-image-preamble> is used to set up the preamble that should be used for making SVG images from your PreTeXt source. Macros that you wish to use more broadly should be put inside <macros> inside <docinfo>.
